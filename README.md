## Development

```bash
$ git clone https://github.com/kicss/kicss.git
$ cd kicss
$ git checkout -b patch-1
$ npm install && gulp build
```

Build -> browserSync,watch
```bash
$ gulp build
```

browserSync, watch
```bash
$ gulp
```
And finally, open in your browser.
