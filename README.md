## Development

Install : `nodejs` `npm` `gulp` `bower` `hugo`
```bash
$ git clone https://github.com/sircus/sircus.git
$ cd sircus
$ git checkout -b patch-1
$ npm install && gulp build
```

Build -> browserSync,watch
```bash
$ gulp build
```

browserSync, watch
```bash
$ gulp
```
And finally, open in your browser.
